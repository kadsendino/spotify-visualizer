# spotify-visualizer

## Description

A terminal-based Spotify visualizer written in Rust for Kitty.

## Installation

### Arch Linux

```
pacman -U spotify-visualizer-bin-0.1.0-1-x86_64.pkg.tar.zst
```

### Tar.gz

```
tar -xzf spotify-visualizer-0.2.0-x86_64.tar.gz
./spotify-visualizer
```

### Build From Source

```
git clone https://github.com/kadsendino/spotify-visualizer.git
cd spotify-visualizer
cargo run --release
```
