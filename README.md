# spotify-visualizer

## Description

A terminal Spotify visualizer written in Rust for Kitty.

## Installation

### Arch Linux

1. Install from AUR

```
paru -S spotify-visualizer-bin
```

2. Download from Github Releases

```
pacman -U spotify-visualizer-bin-0.1.0-1-x86_64.pkg.tar.zst
```

## Build From Source
```
git clone https://github.com/kadsendino/spotify-visualizer.git
cd spotify-visualizer
cargo run --release
```
