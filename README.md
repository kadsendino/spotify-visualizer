# spotify-visualizer

## Description

A terminal Spotify visualizer written in Rust for Kitty.

## Installation

### Arch Linux

1. Install from AUR

```
paru -S spotify-visualizer-bin
```

1. Download from Github Releases

```
pacman -U spotify-visualizer-bin.V.V.V-x86_64.pkg.tar.zst
```
